# Lab1 - Programmation avec l’API HDFS

Ce projet Maven Java (groupId: edu.supmti.hadoop, artifactId: lab1) est destiné aux exercices de programmation avec l’API HDFS.

## Structure du projet
- Répertoire racine : `BigdataLabs/lab1`
- Build tool : Maven (no archetype)
- Java version : 1.8

## Extensions recommandées pour VS Code
- Maven for Java
- Extension Pack for Java

## Dépendances
Ajoutez les dépendances nécessaires à Hadoop ou autres dans le fichier `pom.xml`.

## Compilation
Utilisez la commande suivante pour compiler le projet :

```
mvn clean install
```

## Exécution
Ajoutez votre code source dans `src/main/java` et vos tests dans `src/test/java`.
